В этом проекте по автоматизации тестов я использовал Python, Selenium pytest, POM паттерны и CI/CD через GitHub Actions. Объектом тестирования был сайт Saucedemo.com. Тестовые сценарии покрывают: авторизацию (успешный вход, неверный пароль, заблокированный пользователь, пустые поля), сортировку товаров, добавление в корзину и оформление заказа.

pytest test_POM.py -v
